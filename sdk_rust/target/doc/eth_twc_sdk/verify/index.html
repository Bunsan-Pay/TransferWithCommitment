トランザクションレシート上の TransferWithCommitmentSent イベントの取得・検証。

信頼できる RPC から得たレシートを前提に、指定フィールドと一致するログの有無を確認します（チェーン最終性や悪意ある RPC の改ざんまでは防ぎません）。
