署名付き transferWithAuthorization と cancelAuthorization。

送信前に crate::utils::assert_provider_supported_chain と、署名バンドルの Eip712Domain がプロバイダのチェーンおよび crate::config::transfer_with_commitment_address が返すアドレスと一致することを検証します。 ブロードキャスト前に .call() 相当の eth_call シミュレーションを行い、失敗時は送信しません（sdk_js の simulateContract → writeContract と同様）。
