送金引数・署名済みバンドル・明細行の型定義。

  • args — 自己送金用と EIP-712 署名入力用の引数構造体。
  • signed_data — オフチェーンで渡す署名付きバンドル（正規化済み Eip712Domain を含む）。
  • transfer_detail — 単一行の送金明細。
