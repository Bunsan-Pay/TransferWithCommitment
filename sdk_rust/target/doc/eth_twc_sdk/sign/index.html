EIP-712 オフチェーン署名。

  • オンチェーン eip712Domain() からドメインを取得し、ゼロ salt は OpenZeppelin の domain separator と整合するよう domain_for_typed_data_sign から除外します。
  • Alloy では [Provider] + [Signer] でドメイン取得と署名の両方に足ります（別クライアントに分ける必要はありません）。
