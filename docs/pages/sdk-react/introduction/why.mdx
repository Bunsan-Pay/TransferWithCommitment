# Why

DApp では **接続状態・チェーン・送信中フラグ**が頻繁に変化します。`eth-twc-sdk-js` をそのままコンポーネントに書くと、`useEffect` とローディング管理が膨らみがちです。

`eth-twc-sdk-react` は **wagmi のクライアント取得**と **TanStack Query のミューテーション／クエリ規約**に公式に沿うことで、UI から **「送る／調べる」操作だけ**に集中できるようにします。
