# Installation

```bash
# Bun
bun add eth-twc-sdk-react eth-twc-sdk-js @tanstack/react-query wagmi viem

# npm
npm install eth-twc-sdk-react eth-twc-sdk-js @tanstack/react-query wagmi viem

# pnpm
pnpm add eth-twc-sdk-react eth-twc-sdk-js @tanstack/react-query wagmi viem
```

**peerDependencies** :

- `react` — ^18 または ^19
- `@tanstack/react-query` — ^5.59 系推奨
- `wagmi` — ^3
- `viem` — ^2

## 互換性

- **React** 18 / 19
- **wagmi** v3
- **TanStack Query** v5
- **配布形態**: 未バンドル — Vite / Next.js などを用いた TypeScript 解決が必要
- **チェーン**: SDK は allowlist を持ちません。`wagmi` の `chains` には利用する viem `chain` をアプリが載せます。
