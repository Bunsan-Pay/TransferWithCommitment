# Why

Ethereum 向け Rust では **Alloy** が型安全なコントラクト境界と RPC を提供します。`eth-twc-sdk` はその上に **TWC 固有の EIP-712・`transfer`・`transferWithAuthorization`・イベント検証**だけを載せ、**余計な抽象は置かない**ことを目指します。

`tokio` 上の **`async`** が前提です。同期 API は提供しません。

- **実践**: [Self-Call チュートリアル](/sdk-rust/guides/tutorial-self-call)、[Executor チュートリアル](/sdk-rust/guides/tutorial-signature-executor)
