# TransferWithCommitment（`TransferWithCommitment.sol`）

`SelfTransfer` と `SignatureTransfer` を統合する最終的な派生コントラクト。新たな関数や状態変数は宣言せず、継承によりすべての機能を外部に公開する。

**継承**: `ITransferWithCommitment`, `SelfTransfer`, `SignatureTransfer`

`ITransferWithCommitment` は `ISelfTransfer` と `ISignatureTransfer` を統合するインターフェースであり、新たなメンバは宣言しない。

---

## API Reference

### `constructor`

```solidity
constructor(string memory name, string memory version) SignatureTransfer(name, version)
```

| 引数 | 型 | 説明 |
| --- | --- | --- |
| `name` | `string` | EIP-712 ドメインの `name` フィールド |
| `version` | `string` | EIP-712 ドメインの `version` フィールド |

- **前提条件**: デプロイ時に呼び出し可能であること
- **事後条件**: `SignatureTransfer(name, version)` 経由で `EIP712(name, version)` が設定される。`SelfTransfer`・`State`・`ReentrancyGuard` のデフォルトコンストラクタも実行される。
- **アクション**: `SignatureTransfer(name, version)` を実行する
