# Contracts（`contracts/`）

**ERC-20 の送金**に **任意の `bytes32` commitment** を結びつけ、**`TransferWithCommitmentSent`** イベントでオンチェーンに記録するラッパー実装です。送金経路は **`SelfTransfer`**（`msg.sender` が直接送金）と **`SignatureTransfer`**（EIP-712 署名を Executor が実行）に分かれます。

用語やプロトコル要件の前提は、先に **[Fundamentals](/architecture/fundamentals)**（または **[Architecture](/architecture)** 全体）を読んでおくと、以下の各ページをスムーズに読めます。
